 # 🚆 Train Journey Duration Predictor

This project uses Machine Learning to predict train journey duration based on distance and number of stops.

## 🔧 Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## 🚀 Features
- Data analysis and visualization
- Linear Regression model
- Interactive web app

## ▶️ How to Run
1. Install dependencies:
   pip install pandas scikit-learn streamlit

2. Run the app:
   streamlit run app.py

Shows predicted journey duration based on user input.
